# "Vibe' The Audio Player
Vibe is a light audio player developed in C++ using the JUCE framework with
some easy-to-use and user-friendly features that makes it an enjoyable experience
listening to your favourite tracks.
It functions both as a normal desktop audio player and a simple mixer where you
can mix and play two audio tracks simultaneously.


# Technologies used
(C++ 17)
  ○ Underlying programming language for all logic.
JUCE Framework
  ○ Comprehensive library for Audio I/O, GUI, and more
